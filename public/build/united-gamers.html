  • United Gamers

    Conceptual Prototype | Community Website

    Google Maps, Geolocation, Directions

    For my Design in Subcultures module at RMIBLR, I chose to cover 'gamers'. The analysis & presentations that ensued shed light on the lack of communication between game lounge owners & gamers. To target this issue, I built a prototype community website meant to show game lounges, from a curated list, around a user's location using Google maps. Archived here. (2012-13)

  • Overview

    This project brought me into close contact with cross-browser geo-location, Google maps and most importantly, connecting the two. As the target audience was gamers on the move, the project also included responsive design as another key component, to allow easy access across devices

    Additional features added to the website:

    • Information pop-up for game lounges
    • Directions to chosen game lounge
    • Fully responsive
    • Developed with Twitter's Bootstrap framework
    • Successful linking to test Pushover app
